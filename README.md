# Credit-Card-Customer-Analysis-Dashboard
## Project Objectives
Credit Card Customer Analysis Dashboard:
Built an interactive Power BI dashboard to analyze credit card customer behavior, revenue trends, and demographics. Identified high-value customer segments using KPIs like revenue, transaction amount, interest earned, and customer satisfaction score.

## Questions (KPIs)

- Which gender contributes more revenue?
- Which age group is most valuable?
- How does income level affect spending?
- Which job category contributes the most revenue?
- Which card type is most popular?
- How does marital status impact spending?
- Does education level influence revenue?

## Process

- Collected and cleaned credit card customer data  
- Created data models and defined KPIs using DAX  
- Designed interactive Power BI visuals  
- Analyzed revenue, transactions, and customer demographics  
- Identified spending patterns and high-value customer segments

## Project Insights

- Male customers contribute slightly higher revenue compared to female customers.
- Customers in the 50â€“60 age group generate the highest revenue, indicating stable spending behavior.
- Medium and high-income customers contribute significantly more to total spending.
- Business professionals and white-collar job categories are the top revenue contributors.
- Blue card holders form the largest customer base, while premium cards show higher value per transaction.
- Married customers generate more revenue than single customers.
- Customers with graduate and post-graduate education levels show higher spending patterns.

## Final Conclusion

This project provides a comprehensive analysis of credit card customer behavior using interactive Power BI dashboards. By analyzing revenue, transactions, and customer demographics, the dashboard helps identify high-value customer segments and key revenue drivers. The insights generated can support data-driven decision-making, customer targeting, and revenue optimization strategies for financial institutions.
